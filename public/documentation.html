Movie API Documentation

The API is still under development. The following features will be supported upon release:

  • The API will be a Node.js and Express application.
  • The API will use REST architecture, with URL endpoints corresponding to data operations.
  • The API will use middleware modules, such as the body-parser package for reading data from requests and morgan for logging.
  • The API will use a “package.json” file.
  • The database will be built using MongoDB.
  • The business logic will be modeled with Mongoose.
  • The API will provide movie information in JSON format.
  • The JavaScript code will be error-free.
  • The API will be tested in Postman.
  • The API will include user authentication and authorization code.
  • The API will include data validation logic.
  • The API will meet data security regulations.
  • The API source code will be deployed to a publicly accessible platform like GitHub.
  • The API will be deployed to Heroku.
